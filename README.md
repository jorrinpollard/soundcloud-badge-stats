# soundcloud-badge-stats

Displays followers and tracks below user badges on SoundCloud

## Requirements

- Firefox 57.0+

## Installation

- [Download at addons.mozilla.org](https://addons.mozilla.org/en-US/firefox/addon/soundcloud-badge-stats/)
